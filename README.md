# 🎥 YouTube Video Chatbot using RAG

A powerful chatbot that enables interactive conversations with YouTube video content using Retrieval-Augmented Generation (RAG) and LangChain.

## 🎯 Overview

This project implements an intelligent chatbot that allows users to interact with YouTube videos using natural language queries. Instead of manually watching entire videos, users can ask questions and get relevant answers extracted from the video content. The system leverages RAG (Retrieval-Augmented Generation) to retrieve contextual information from videos and provide accurate, grounded responses.

## ✨ Features

- **YouTube Video Summarization**: Automatically extract and summarize key information from YouTube videos
- **Natural Language Queries**: Ask questions in plain English about video content
- **Context-Aware Responses**: Get accurate answers based on actual video content using RAG
- **LangChain Integration**: Powered by LangChain for advanced language model operations
- **Interactive Chat Interface**: Conversational interaction with video content
- **Multi-Query Support**: Handle complex questions requiring information synthesis from multiple parts of videos
- **Easy Integration**: Simple API for integrating into existing applications

## 🛠️ Tech Stack

- **Python 3.8+**: Core programming language
- **LangChain**: Framework for building language model applications
- **RAG (Retrieval-Augmented Generation)**: Technique for grounded information retrieval
- **YouTube API**: For accessing video content and metadata
- **Jupyter Notebook**: Interactive development environment
- **Vector Databases**: For efficient document retrieval
- **Language Models**: Integration with state-of-the-art LLMs
